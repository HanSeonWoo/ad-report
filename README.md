# 광고 정산 데이터 리포트 프로젝트

이 프로젝트는 광고 정산 데이터를 받아와 해당 데이터를 통해 제공할 수 있는 다양한 형태의 리포트를 보여주는 React 프로젝트입니다.

## 기술 스택

- **프레임워크**: React.js
- **빌드 도구**: Vite
- **스타일링**:
  - Tailwind CSS
  - shadcn (UI 컴포넌트 라이브러리)

## 실행 방법

프로젝트를 실행하려면 다음 명령어를 사용하세요:

```bash
yarn && yarn dev
```
